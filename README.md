Requerimentos:
- é necessário ter o Node.js instalado

Para testar:
- npm test

Para executar:
- node server

Para executar em cluster:
- npm run clusters

Para gerar a documentação e executar:
- npm start
- a documentação pode ser acessada através de /apidoc
